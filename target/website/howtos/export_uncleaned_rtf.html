Questa guida spiega come produrre un file in formato "RTF non ripulito" da un progetto OmegaT. Questo può essere un pratico formato di interscambio per gli utenti di altre applicazioni CAT, in particolare Trados e Wordfast Classic.

Punti da tenere presente:

Questa combinazione di codice/procedura non è stata ancora accuratamente testata (anche se i risultati iniziali sono promettenti).

Lo script è disponibile solo per la piattaforma Windows.

Permette ai traduttori di generare (cioè consegnare) solo file Trados non ripuliti. Non permette loro di "tradurre" file bilingui forniti dai clienti.

1. Inserire i delimitatori di segmento di Trados

Per produrre un file di LibreOffice/OpenOffice.Org bilingue contenente i delimitatori di segmento di Trados, chiudere il progetto OmegaT, quindi eseguire il seguente script AutoIt (richiede Windows) sul file del progetto project_save.tmx:

www.leuce.com/autoit/uncleanifyTMX.zip

2. Creare documenti di destinazione

Creare i documenti di destinazione come al solito e convertire il file in Microsoft Word.

3. Applicare stili ai delimitatori

In Microsoft Word, eseguire la seguente macro per applicare gli stili pertinenti ai documenti di destinazione:

Private Sub AddTw4winStyles()
On Error Resume Next
Dim sMark, sInt, sExt As Style
Set sMark = ActiveDocument.Styles.Add(Name:="tw4winMark", _
    Type:=wdStyleTypeCharacter)
With sMark.Font
  .Name = "Courier"
  .Size = 11
  .Hidden = True
  .Subscript = True
  .ColorIndex = wdViolet
End With
Set sInt = ActiveDocument.Styles.Add(Name:="tw4winInternal", _
    Type:=wdStyleTypeCharacter)
sInt.LanguageID = wdNoProofing
With sInt.Font
  .Name = "Courier"
  .Size = 11
  .ColorIndex = wdRed
End With
Set sExt = ActiveDocument.Styles.Add(Name:="tw4winExternal", _
    Type:=wdStyleTypeCharacter)
sExt.LanguageID = wdNoProofing
With sExt.Font
  .Name = "Courier"
  .Size = 11
  .ColorIndex = wdGray50
End With
End Sub
